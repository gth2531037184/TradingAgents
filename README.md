# MT5 高频交易系统

一个基于 MetaTrader5 (MT5) 的高频交易系统，使用头皮交易（Scalping）策略进行快速交易。

## 📋 功能特性

- ✅ **快速交易**：快进快出，及时获利了结
- ✅ **自动止盈止损**：预设盈利目标 0.1 美元，止损 0.1 美元
- ✅ **黄金交易**：专注 XAUUSD（黄金）品种
- ✅ **灵活配置**：交易量、交易时间可调整
- ✅ **风险管理**：日亏损限制、持仓管理
- ✅ **实时监控**：显示账户、持仓、交易统计信息
- ✅ **日志记录**：所有交易都有详细的日志记录

## 🚀 快速开始

### 第一步：安装 Python

下载并安装 Python 3.8+ 版本：
https://www.python.org/downloads/

### 第二步：安装必要的库

打开命令行，输入以下命令：

```bash
pip install MetaTrader5
pip install pandas
pip install numpy
```

### 第三步：安装 MT5 平台

1. 下载 MetaTrader5：https://www.metatrader5.com/
2. 注册账户（选择模拟盘或真实账户）
3. 启动 MT5 平台

### 第四步：配置系统

编辑 `config.py` 文件，填入你的 MT5 账户信息：

```python
ACCOUNT_INFO = {
    "login": 你的账户号,              # 例如: 123456789
    "password": "你的密码",           # 例如: "password123"
    "server": "你的服务器",           # 例如: "MetaQuotes-Demo"
}
```

### 第五步：运行程序

```bash
python main.py
```

## ⚙️ 配置说明

### 交易参数（`config.py`）

```python
TRADING_CONFIG = {
    "symbol": "XAUUSD",           # 交易品种：黄金
    "lot_size": 0.1,              # 交易量：0.1手（可调整）
    "take_profit": 0.1,           # 盈利目标：0.1美元
    "stop_loss": 0.1,             # 止损点数：0.1美元
    "max_positions": 1,           # 最大持仓数
    "max_daily_loss": 50,         # 日最大亏损额（美元）
}
```

### 交易时间配置

```python
TRADING_HOURS = {
    "enabled": True,              # 是否启用时间限制
    "start_hour": 0,              # 开始时间（24小时制）
    "end_hour": 23,               # 结束时间
}
```

## 🎯 交易策略

### 头皮交易（Scalping）

这个系统使用简单的头皮交易策略：

1. **信号生成**：
   - 价格上升 → 买入信号
   - 价格下降 → 卖出信号
   - 无明显方向 → 不交易

2. **风险控制**：
   - 自动止盈：达到 0.1 美元利润时自动平仓
   - 自动止损：亏损 0.1 美元时自动平仓
   - 日亏损限制：亏损超过 50 美元时停止交易

3. **仓位管理**：
   - 最多只持有 1 个仓位
   - 避免过度交易

## 📊 监控和日志

程序会生成详细的日志文件 `trading_log.txt`，包含：

- 连接状态
- 账户信息
- 每个交易的详细信息
- 交易统计
- 风险管理状态
- 错误信息

### 常见问题

**Q: 如何连接到我的 MT5 账户？**
A: 编辑 `config.py`，填入你的账户号、密码和服务器地址。

**Q: 如何调整交易量？**
A: 编辑 `config.py` 中的 `lot_size` 参数。

**Q: 如何修改盈利目标和止损？**
A: 编辑 `config.py` 中的 `take_profit` 和 `stop_loss` 参数。

**Q: 程序运行期间没有交易怎么办？**
A: 检查：
1. MT5 是否在运行且成功连接
2. 当前时间是否在交易时间内
3. 查看日志文件 `trading_log.txt` 了解详细信息

**Q: 如何只在特定时间交易？**
A: 编辑 `config.py` 中的 `TRADING_HOURS` 参数。
